socket编程

2020-04-27- 2020-05-08

最近学习了用socket编程实现TCP通信，简单归纳。

常用函数

WSAStartup()

函数原型：WSAStartup(WORD wVersionRequested, LPWSADATA lpWSAData);
使用方法：WORD参数由MAKEWORD(a,b)创建，其中a,b为版本号，LPWSADATA 是一个WSADATA结构的指针
返回值：成功返回0，各类错误返回值可查看说明
注意：此函数是初始化Winsock服务，且在最后需要使用WSACleanup()，在linux则中不需要使用。

socket()

函数原型：int socket(int domain, int type, int protocol);
使用方法：domain包括 AF_INET(IPv4因特网域)， AF_INET6(IPv6因特网域), AF_UNIX(UNIX域)， AF_UNSPEC(未指定)。type包括 SOCK_DGRAM(长度固定的、无连接的不可靠报文传递), SOCK_RAW(IP协议的数据报接口(POSIX.1中为可选)), SOCK_SEQPACKET(长度固定、有序、可靠的面向连接报文传递), SOCK_STREAM(有序、可靠、双向的面向连接字节流)。protocol通常是零，表示按给定的域和套接字类型选择默认协议。
返回值：成功返回套接字描述符，错误返回-1

bind()

函数原型：int bind(int sockfd, const struct sockaddr *addr, socklen_t len);
使用方法：sockfd是用socket()生成的套接字描述符，addr是一个指向sockaddr结构体类型的指针，len表示addr结构的长度，可用sizeof()获得。
返回值:成功返回0，错误返回-1

listen()

函数原型：int listen(int sockfd, int backlog);
使用说明：sockfd使用socket()创建的套接字描述符,backlog是等待连接队列的最大长度
返回值：成功返回0，错误返回-1

accept()

函数原型：int accept(int sockfd, struct sockaddr *reStrictaddr addr, socklen_t *restrict len);
使用说明：sockfd使用socket()创建的套接字描述符,addr用于接收连接实体的地址，len为addr的长度
返回值：成功返回套接字描述符，错误返回-1

connect()

函数原型：int connect(int sockfd, const struct sockaddr *addr, socklen_t len);
使用说明：sockfd是使用socket()创建的套接字描述符,addr是sockaddr结构的指针，表示服务端地址，len为addr的长度
返回值：成功返回0，错误返回-1

send()

函数原型：ssize_t send(int sockfd, const void *buf, size_t nbytes, int flags);
使用说明：sockfd是使用socket()创建的套接字描述符,buf是一个存放要发送数据的缓冲区，nbytes指明实际要发送的数据的字节数,flags一般置零
返回值：若成功则返回发送的字节数，若出错则返回-1

recv()

函数原型：ssize_t recv(int sockfd, void *buf, size_t nbytes, int flags);
使用说明：sockfd是使用socket()创建的套接字描述符,buf是一个存放要接收数据的缓冲区，nbytes指明缓冲区的长度,flags一般置零
返回值：以字节计数的消息长度，若无可用消息或对方已经按序结束则返回0，若出错则返回-1

close()

函数原型：int close(int sockfd)
使用说明：用来关闭套接字，并终止TCP连接
返回值：成功返回0，错误返回-1
