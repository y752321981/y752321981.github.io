在学了在学了之H5(day 1)

2020-04-28- 2020-05-09

入门

<!doctype html>
这是我的第一个网页 这是我的第一个HTML5网页！

<!doctype html>

只有开头用此标记声明，浏览器才能将该网页作为有效的HTML文档

<html></html>

标记用于<!doctype html>之后，被称为根标记

<head></head>

标记用于定义HTML的头部信息，紧跟在<html>标记之后，主要用来封装其他位于文档头部的标记，例如<title>、<meta>、<link>及<style>，用来描述文档的标题、作者以及和其他文档的关系

<body></body>

标记用于定义HTML文档所要显示的内容，标记中的信息才是最终展示给用户看的。

标题和段落标记、网页文本

标题标记<hn></hn>

HTML提供了6个等级的标题，<h1><h2><h3><h4><h5><h6>，依次递减的黑字体显示标记，语法

<hn align="对齐方式">标题文本</hn>

n为1到6的取值，align为可选属性left/right/center

段落标记<p></p>

<p align="对齐方式">段落文本</p>

align为可选属性left/right/center

水平线标记<hr />

<hr 属性="属性值" />

常用属性

属性名 含义 属性值
align 设置水平线的对齐方式 可选择left、right、center三种，默认为center
size 设置水平线的粗细 以像素为单位，默认2像素
color 设置水平线的颜色 可用颜色名称、十六进制#RGB、rgb(r,g,b)
width 设置水平线的宽度 可以是确定的像素值，也可以是浏览器窗口的百分比，默认为100%

换行标记< br />

<br />

文本格式化标记

标记 显示效果
和 文本以粗体方式显示（b定义文本粗体,strong定义强调文本）
和 文本以斜体方式显示（i定义斜体字，em定义强调文本）
和 文字以加删除线方式显示（HTML5不赞成使用s）
和 文字以加下划线方式显示（HTML5不赞成使用u）

特殊字符标记

空格 &nbsp

>,< &lt,&gt;

n次方 &supn

图片设计

图像标记<img />

<img src="图像URL及图片名称" />

常用 gif,png,jpg格式

属性 属性值 描述
src URL 图像的路径
alt 文本 图像不能显示时的替换文本
title 文本 鼠标悬停时显示的内容
width 像素（XHTML不支持%页面百分比） 设置图像的宽度
height 像素（XHTML不支持%页面百分比） 设置图像的高度
border 数字 设置图像边框的宽度
vspace 像素 设置图像顶部和底部的空白（垂直边距）
hspace 像素 设置图像左侧和右侧的空白（水平边距）
align left 将图像对其到左边
right 将图像对其到右边
top 将图像的顶端和文本的第一行文字对齐，其他文字居图像下方
middle 将图像的水平中线和文本的第一行文字对齐，其他文字居图像下方
bottom 将图像的底部和文本的第一行文字对齐，其他文字居图像下方

绝对路径

绝对路径一般指带有盘符的路径，网页中不推荐使用绝对路径。

相对路径

通常以HTML网页文件为起点，通过层级关系描述目标图像的位置

超链接

创建超链接

<a href="跳转目标" target="目标窗口的弹出方式"> 文本或图像</a>

href:用于指定链接目标的url地址，当a标记应用href属性时，他就具有了超链接的功能。

target:用于指定链接页面的打开方式，其取值有_set和_blank两种，其中_self为默认值，意为在原窗口中打开，_blank在新窗口中打开。
