工作空间与使用帮助

  • clear:清除工作空间中的所有变量

  • clear 变量名：清除指定的变量

  • 永久常数：

    • pi：圆周率π
    • inf：无限大
    • ans：用于显示结果的缺省变量名

  • save filename variables保存变量在文件filename.mat中

  • load filename variables调用文件filename.mat中的变量到MATLAB工作空间中
  • help ‘function name’：查询该函数的用法，比如help inv是用来查询计算反矩阵的用法
  • helpwin：帮助窗口
  • demo:示例窗口，MATLAB有丰富的示例供学习参考

基本运算

书写向量（vectors）与矩阵（matrices）

  • ```matlab
    t=[5:-1:2] %产生行向量,从5到2间隔为-1
    x=[1 2 3 -4 -5] %或x=[1,2,3,-4,-5]表示行向量
    x=[1;2;3;-4;-5] %用分号表示列向量
    输入矩阵：A=[1.2 10 15;3 5.5 2;4 6.8 7]
    如果在矩阵A后面加上分号;取消打印
    A(i,:)第i行,A(:,j)第j列
    y=x’ %表示x的反转矩阵为y

  • diag(n:m)n≤m，产生一个对角元素为从n到m的m-n+1阶对角矩阵，若n=m则产生一个元素为n的一阶矩阵

时间与日期

  • clock：产生包含年、月、日、时、分、秒的行向量
  • date：当前日期

加法与减法（addition and subtraction)

  • A=[2 3;4 5;6 7];B=[1 0;2 3;0 4];C=A+B

  • x=[5;4;6];y=x-1

乘法（multiplication）

  • ```matlab
    x=[1;2;3]
    y=[4;5;6]
    z=x’y
    u=x    y’
    v=y*x’

数组的幂

同数组相乘一样，对矩阵或向量的元素求幂，表示为.^

绝对值（absolute values)

复数的绝对值等于实部平方与虚部平方的和再开平方根

数组除法（Array division）

  • 数组相除为左除./与右除. A除以B A./B B除以A A.\B

  • 除法中要注意分母为0时，MATLAB会给出警告

矩阵函数

  • 特征值与特征向量（Eigenvalues and eigenvectors）

    • A是一个n×n矩阵，具有n个特征值满足

      特征值λ在MATLAB中的计算函数为

      ​ eig(A) %产生一个列向量表示特征值

      如果A是实数并且对称的，那么特征值为实数，如果A不是对称的，特征值经常是复数：

      A=[0 1 0;-1 0 2;3 0 5];eig(A) %A矩阵，求A的特征值

    • 根据公式

      这里D为矩阵形式的特征值，X为矩阵形式的特征向量

      A=[0 1 0;0 0 1;-6 -11 -6];[X D]=eig(A)

  • 特征方程（caracteristic equation）

    • 矩阵的特征方程的根即为矩阵的特征值

      得到p=[1 6 11 6]表示特征方程的系数，特征方程为：

      计算特征方程p的根：

      r=roots(p)

      计算矩阵A的特征值

      lameda=eig(A)

      两者的相等的

    • 还可以已知多项式的根，用函数poly(r)求多项式

  • 多项式乘积（卷积,convolution）

    将两个多项式系数向量相乘 conv(a,b)

  • 多项式相除（解卷积，deconvolution）

    将两个多项式系数向量相除 deconv(a,b)

    q和r是多项式相除得到的商和余数

    得到q=[3 4 -13]，r=[0 0 0 22 180]

    即a(s)=b(s)*q(s)+r(s)

绘图功能

  • 二维线型图形：plot

    比如，定义时间从0到2π，每隔0.01π取一个时间点，在该时间序列内计算正弦函数y=sin(3*t)的数值，并以时间为x轴，以函数值为y轴，画出时间响应曲线：

  • 添加网格线，图形标题，x轴标题，y轴标题：

  • 在图形上标记文字：

  • 限制x、y轴区域

    • 在MATLAB绘图中，x轴和y轴的区域是自动选择的，有时需手动选择区域
    • xlim([0 2*pi]);ylim([-2 2]); %限定x轴和y轴的范围

  • 选择图形形状

    • 普通默认为实线图，即将各点用曲线连起来，可以选择其他形状绘图

    • 线绘图包括“-实线”，“—短划线”，“:点线”，“-.短划线与点线”：

      plot(t,y,'-',t,2*y,'--',t,3y,':',t,4y,'-.')

    • 点绘图包括”+”，”o”，”*”，”x”等，详见help plot

      plot(t,y,'-',t,2*y,'--',t,3y,':o',t,4y,'-.x')

  • 选择图形颜色

    ​ 系统默认颜色为蓝色，可以自己选择图形颜色：

    ​ red r; green g; blue b

    ​ white w; yellow y; magenta m(紫色)

    ​ cyan c;(青色)black b

    ​ plot(t,y,'-r',t,2*y,'--+c',t,3y,':om',t,4y,'-.xg')

  • 选择线的粗细大小

    ​ 系统默认为0.5，可以自己加粗线：

    ​ plot(t,y,'-r',t,2*y,'--+c',t,3*y,':om',t,4*y,'-.xg','LineWidth',3)

  • 在图形上显示希腊字符与符号

    • 显示希腊字符

      α \alpha; β \beta;

      γ \gamma; Γ \Gamma;

      δ \delta; Δ \Delta;

      ω \omega; Ω \Omega;

      σ \sigma; Σ \Sigma;

      Φ \phi; ψ \psi;

      θ \theta; ζ \zeta;

      π \pi; Π \pi

      plot(t,y);text(pi,sin(3*pi),'sin(3\pi)');

    • 显示其他数字符号：

      ​ ∞ \infinity ← \leftarrow

      ​ ↑ \uparrow → \rightarrow

      ​ ↓ \downarrow ° \circ

      ​ ± \pm

      plot(t,y);text(pi,sin(3*pi),'\leftarrowsin(3\pi)');
