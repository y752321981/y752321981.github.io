连续系统模型

多项式传递函数

  • 可以将函数写成分子（numerator）、分母（denominator）两个多项式，每个多项式中的系列作为行向量在MATLAB中写出

    num=[b0 b1 ... bn]

    den=[1 a1 ... an]

    sys=tf[num,den] 给出以num为分子、den为分母的传递函数

    如：

  • 使用分式相除指令residue可以将传递函数进行部分分式展开：

    [r,p,k]=residue(num,den)

    r表示分子，p表示分母中的极点，k表示剩余量。

    num=[b0 b1 ... bn]

    den=[1 a1 ... an]

    r=[-6 -4 3]

    p=[-3 -2 -1]

    k=2

零极点传递函数

  • 零极点形式的传递函数首先要已知三个数值：

    • 零点的行向量，z=[…]

    • 极点的行向量，p=[…]

    • 增益值，k

      z=[-3 -1]; p=[0 -2 -4 -6]; k=4

      sys2=zpk(z,p,k)

  • 零极点传递函数转化为多项式传递函数：

    systf=tf(sys2)

  • 多项式传递函数转化为零极点传递函数：

    syszpk=zpk(systf)

多输入多输出的传递函数

  • 2输入2输出的传递函数

    H=tf({-5;[1 -5 6]},{[1 -1];[1 1 0]})

  • 转化为零极点形式：H_zpk=zpk(H)

    用零极点形式输入：

    H_zpk=zpk({0;[3 2]},{1;[0 -1]},[-5;1])

状态方程

  • sys=ss(A,B,C,D)产生一个连续时间状态空间模型，矩阵参数为给定的A B C D

模型转换

  • 以上提到的转换函数只给出了模型打印形式，并没有给出具体的A B C D或num、den、zpk具体数值，为了方便与分析系统的特性，下面给出三种模型的转换方法：

    • 传递函数转换为状态方程：

      [A,B,C,D]=tf2ss(num,den)

    • 传递函数转换为零极点形式：

      [z,p,k]=tf2ss(num,den)

离散时间系统模型

  • sysd=c2d(sys,Ts,method)

    连续系统为sys，Ts为采样时间，method为采用的保持器，默认为零阶保持器，’zoh’，默认情况下可以去掉method

    还有三角形近似和双线性近似：’for’’tustin’

  • [G,H]=c2d(A,B,Ts0)

    A和B为状态方程G和H为离散化状态方程矩阵

矩阵，Ts采样时间，默认零阶保持器，输出
